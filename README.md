Tampilan Sistem – Halaman Login
Halaman Login merupakan halaman awal yang pertama kali muncul saat pengguna mengakses website Toko Handphone Gen Z melalui browser.
Halaman ini berfungsi untuk autentikasi pengguna, baik sebagai Admin maupun User, sebelum masuk ke sistem utama.

Tampilan Sistem – Halaman Admin Handphone
Halaman Admin Handphone Gen Z merupakan halaman utama yang ditampilkan setelah admin berhasil login.
Halaman ini berfungsi untuk mengelola data handphone yang ada di dalam sistem.
Admin dapat melihat seluruh produk handphone beserta detailnya, serta melakukan pengelolaan data melalui fitur CRUD.
